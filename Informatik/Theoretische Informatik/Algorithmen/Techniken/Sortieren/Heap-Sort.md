# Heap-Sort #card
## Funktionsweise
Nimmt eine Folge von $n$ vergleichbaren Elementen entgegen und bildet aus ihnen einen [[Binary-Heap|Min-Heap]] in $\Theta(n)$. Im Anschluss werden alle $n$ Elemente in jeweils $O(\log n)$ aus dem Heap entfernt. Der [[Binary-Heap|Min-Heap]] garantiert das immer das kleinste noch im Heap vorhandene Element entfernt wird.

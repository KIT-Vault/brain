# Zwischenwertsatz
## Definition
Sei $f$ eine [[Stetigkeit|stetige]] [[Funktion]] mit $a < b$ und $f(a) < f(b)$, so folgt nach Mittelwertsatz, dass es ein $x_{0}$ gibt mit $f(x_{0}) = y_{0}$.
